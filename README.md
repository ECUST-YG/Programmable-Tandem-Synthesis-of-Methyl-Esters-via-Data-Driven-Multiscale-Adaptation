This repository contains the code for the paper ‘Programmable Tandem Synthesis of Methyl Esters via Data-Driven Multiscale Adaptation’. A dual Cu-Cs tandem catalyst has recently been identified for methyl acrylate (MA), methyl propionate (MP) and methyl methacrylate (MMA) synthesis. However, its rational and efficient design remains challenging because of the inherent multiscale coupling in the tandem catalytic system. To address this issue, we propose a data-driven multiscale adaptation strategy for Cu-Cs tandem catalysis. CNN model: The ‘CNN model’ folder contains the data and code used for model construction and prediction. The CNN model is built from structural matrices to learn the relationship between spatial organization and acid-base properties. In particular, ‘Original_database’ provides the dataset used for the ‘Model construction’ code. ‘Cs-Si_61421’, ‘Ti-Cs_59917’, and ‘Ti-Si_61285’ are input datasets for the ‘Prediction’ code. The predicted high-performance catalyst features Ti–O–Cs structures with a Ti/Cs molar ratio close to 1:1 and a moderate Ti-Cs dispersion. Neural ODEs model: The ‘Neural ODEs model’ folder contains the code used for catalyst bed design and product yield calculation. ‘model_params_MA’, ‘model_params_MP’, and ‘model_params_MMA’ are utilized by the ‘Neural_ODEs_MA’, ‘Neural_ODEs_MP’, and ‘Neural_ODEs_MMA’ code, respectively. The optimized catalyst bed profile in terms of axial distribution of Cu and Ti-Cs catalysts can be obtained.
